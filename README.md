🎨 Thumblify
🚀 AI-Powered Thumbnail Generator (MERN Stack)








✨ Overview

Thumblify is a full-stack MERN application that generates eye-catching thumbnails from user prompts using AI. It helps content creators quickly turn ideas into visually appealing thumbnails without manual design.

⚡ Features
🧠 AI-based thumbnail generation from text prompts
🎯 Clean and intuitive UI
🔐 Authentication system (if implemented)
⚡ Fast and responsive application
📦 Full-stack MERN architecture
🛠️ Tech Stack
💻 Frontend
React.js
Tailwind CSS / CSS
Axios
🖥️ Backend
Node.js
Express.js
🗄️ Database
MongoDB (Mongoose)
🤖 AI Integration
Gemini API / OpenAI
📁 Project Structure

⚙️ Installation & Setup
1️⃣ Clone Repository
git clone https://github.com/ankit20092005/thumblify.git
cd thumblify
2️⃣ Backend Setup
cd server
npm install

Create .env file inside server:

PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret
GEMINI_API_KEY=your_api_key

Run backend:

npm run dev
3️⃣ Frontend Setup
cd client
npm install
npm start
🚀 Usage
Enter a prompt
Generate thumbnail using AI
Download or use it
