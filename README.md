# 🎨 Thumblify  
### 🚀 AI-Powered Thumbnail Generator (MERN Stack)

## ✨ Overview

Thumblify is a full-stack MERN application that generates eye-catching thumbnails from user prompts using AI.  
It helps content creators quickly turn ideas into visually appealing thumbnails without manual design.

---

## ⚡ Features

- 🧠 AI-based thumbnail generation from text prompts  
- 🎯 Clean and intuitive UI  
- 🔐 Authentication system (if implemented)  
- ⚡ Fast and responsive application  
- 📦 Full-stack MERN architecture  

---

## 🛠️ Tech Stack

### 💻 Frontend
- React.js  
- Tailwind CSS / CSS  
- Axios  

### 🖥️ Backend
- Node.js  
- Express.js  

### 🗄️ Database
- MongoDB (Mongoose)  

### 🤖 AI Integration
- Gemini API / OpenAI
